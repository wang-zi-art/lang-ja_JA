Hide Deprecated Applications=非推奨アプリを非表示
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=この設定は、利用可能なアプリのリストから非推奨アプリを非表示にします。ただし、非推奨アプリはインストール済みアプリと以前のアプリのセクションには引き続き表示されます
Hide Incompatible Applications=非互換アプリを非表示
This setting will hide applications that are listed as being incompatible with your version of Unraid=この設定は、お使いのUnraidバージョンと非互換としてリストされているアプリを非表示にします
Allow install of second instance=2つ目のインスタンスのインストールを許可
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=このオプションは、カードとポップアップに別のボタンを追加し、デフォルトのテンプレート値を使用してインストール済みアプリを再インストールできるようにします。このオプションは混乱を招く場合があり、通常は同じアプリを同時に2つインストールして実行したい場合にのみ使用されます
Automatically open the sidebar=サイドバーを自動的に開く
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=アプリのアイコンにマウスを合わせたときに、CAがサイドバーを自動的に開いて詳細情報を表示します
Allow CA to check for updates to applications=CAによるアプリの更新確認を許可
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=この設定により、CAがアプリの更新を確認できるようになります。なお、Community Applicationsは常に自身の更新を確認します
Disable Mouse Wheel Page Changes=マウスホイールによるページ変更を無効化
This setting disables using left or right on a mouse wheel to change the page=この設定は、マウスホイールの左右傾斜によるページ変更を無効にします
Mouse wheel threshold=マウスホイールの閾値
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=マウスホイールを左右に傾けると、ブラウザが返す値が変化します。返された値を閾値として設定してください。ブラウザやOSによって異なる値が返される場合があります
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=これは全デバイスに対するグローバル値です。問題が発生する場合は、マウスホイールによるページ変更を無効にしてください
Allow CA to send any emergency notifications=CAによる緊急通知の送信を許可
This setting allows CA to periodically scan your system and alert you to the presence of=この設定により、CAが定期的にシステムをスキャンし、以下の存在について警告します
1 Known malware and security vulnerabilities within applications=1 アプリ内の既知のマルウェアとセキュリティ脆弱性
2 Critical security vulnerabilities within Unraid OS=2 Unraid OS内の重大なセキュリティ脆弱性
3 Any installed applications which may seriously hinder the operation of your server=3 サーバーの動作を深刻に妨げる可能性のあるインストール済みアプリ
Save CA debugging information=CAデバッグ情報を保存
Only enable this when advised to from the author of Community Applications=Community Applicationsの作者から指示があった場合にのみ有効にしてください
Enable developer mode=開発者モードを有効化
This unlocks some extra features within CA generally only used by developers Not recommended to enable=これにより、CA内の開発者向け追加機能のロックが解除されます。有効にすることは推奨されません
